# Job Portal Backend

A REST API built with Node.js, Express and MongoDB.

## Stack
- Node.js + Express
- MongoDB + Mongoose
- JWT for auth
- Nodemailer for emails

## Setup

```bash
npm install
cp .env.example .env
# fill in your values in .env
npm run dev
```
